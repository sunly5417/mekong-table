# Mekong Table - Restaurant Website

A multi-page restaurant website for Mekong Table, a modern dining experience serving authentic Khmer cuisine and Western classics in Phnom Penh, Cambodia.

## Pages

- **Home** - Hero section, featured dishes, about preview, and why choose us
- **Menu** - Full menu listing with Khmer dishes, Western classics, and beverages with prices
- **About** - Restaurant story, values, and team section
- **Contact** - Contact information, message form, and location

## Features

- Responsive design that works on desktop and mobile
- Navigation bar present on all pages linking to all other pages
- Semantic HTML structure throughout
- SEO meta tags on every page
- Clean and consistent CSS styling using CSS variables
- Contact form with multiple input types

## Tech Stack

- HTML5
- CSS3

## Project Structure

```
mekong-table/
├── index.html

```

## How to Run

No installation or setup needed. Simply download all files into one folder and open `index.html` in your browser.

## Pages Overview

### Home
The homepage introduces the restaurant with a full-height hero section, a preview of featured dishes from both Khmer and Western menus, and a section highlighting the restaurant's key values.

### Menu
The menu page is divided into three sections: Traditional Khmer Dishes, Western Classics, and Beverages. Each item includes a name, description, and price.

### About
The about page tells the story of the restaurant, its founder Chef Sovann Keo, the values the restaurant stands for, and introduces the kitchen team.

### Contact
The contact page provides the restaurant address, phone numbers, email, and opening hours alongside a contact form for inquiries and reservations.
